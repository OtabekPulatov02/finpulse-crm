# Finpulse CRM — HTML-прототип

Интерактивный прототип CRM-системы для бухгалтерской компании: учёт клиентов, управление задачами, интеграция с Telegram, аналитика и ролевой доступ.

**Только HTML / CSS / JavaScript** — без backend, API и базы данных. Все данные демонстрационные, взаимодействия имитируются на клиенте.

## Страницы

- `login.html` — экран входа (переход с `index.html`)
- `dashboard.html` — дашборд: KPI, задачи на сегодня, обращения из Telegram, загрузка сотрудников, диаграмма статусов
- `clients.html` / `client.html` — список и карточка клиента (вкладки: обзор, задачи, история, комментарии, файлы)
- `employees.html` / `employee.html` — список и карточка сотрудника
- `tasks.html` / `task.html` — задачи в режимах «Таблица» и «Канбан», карточка задачи
- `telegram.html` — демо потока: сообщение клиента → задача → автораспределение → уведомление
- `analytics.html` — производительность, загрузка, статистика по статусам
- `settings.html` — пользователи, роли, статусы, правила распределения, уведомления

## Ролевой доступ (демо)

Переключатель в правом верхнем углу:

- **Супер-админ** — Ибрагимова Юлдуз Ахмедовна (главный бухгалтер): видит всё.
- **Бухгалтер** — личные данные клиентов скрыты (контакты, телефоны, адреса, тарифы), административные действия и раздел «Настройки» недоступны.

## Технологии

Semantic HTML5, чистый CSS (дизайн-система в `assets/css`), ванильный JS (`assets/js/app.js`), шрифт Inter, иконки Lucide (CDN).

## Запуск

Откройте `index.html` в браузере — сборка не требуется.
